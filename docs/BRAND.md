## 名称

**CBLens** — A 股可转债定价与机会筛选工作台

### 命名由来

| 元素 | 含义 |
| :---: | :--- |
| **CB** | Convertible Bond 的缩写；贴合项目已有的 `CB.py`、`cb-*` 命令 |
| **Lens** | 「透镜 / 棱镜」隐喻——不替用户做判断，而是把条款、公告、行情和模型偏差放大到可复核的视图里 |

### 命名保留

为避免破坏已有脚本和 import，以下技术名称保持不变或按公开名称收敛：

| 上下文 | 名称 |
| :--- | :--- |
| Python 包名 | `convertible_bond` |
| pip 安装名 | `cblens` |
| CLI 入口 | `cb-gui`、`cb-sync-events`、`cb-screen-pool` 等 |

### 视觉元素

图标浓缩了 CBLens 的核心动作：

| 元素 | 象征 |
| :--- | :--- |
| **深色圆角底** | 研究终端与夜间复盘场景 |
| **债券凭证（纸面文档）** | 条款、票息、余额、评级等半静态信息 |
| **凸性曲线（金色弧线）** | 可转债的转股价值、债底与期权价值 |
| **放大镜** | CBLens 的核心动作：筛选、放大异常、交给人工复核 |
| **青绿色镜框** | 主色调，传达分析冷静感 |
| **金色手柄与信号点** | 机会信号——但不画成"买入信号"，避免过度承诺 |
| **CB 徽标** | 文档右上角小标签，点明可转债语境 |

## 使用规范

### 推荐做法

- ✅ README、文档首页、GUI 标题统一写作 **CBLens**
- ✅ 技术上下文中可以写 `convertible_bond` 包；公开安装名统一写 `cblens`
- ✅ 图标优先使用 SVG；需要 macOS `.icns` 或 Windows `.ico` 时，从 SVG 导出 1024px 源图再生成
- ✅ 推荐措辞：「研究工作台」「复核视图」「模型偏差」「风险标签」

### 避免事项

- ❌ 不要把 CBLens 描述成自动交易系统
- ❌ 不要使用「荐股」「收益承诺」「买入信号」等措辞
- ❌ 不要在图标或品牌材料中暗示投资回报保证
