### 正文

最近把自己做 A 股可转债研究时用的一套工具整理成了桌面 App，名字叫 CBLens。

它不是下单软件，也不是投资建议，更像一个“复核工作台”：把转债条款、公告事件、正股行情、历史波动率、信用利差和 PDE 理论定价串起来，最后给出一张适合人工复核的候选表。

我自己最想解决的是这几个问题：

- 全市场转债太多，先用准入筛选排掉停牌、强赎、临近摘牌、ST、低流动性等明显风险。
- 只看溢价率不够，希望能同时看到理论价、市价偏离、转股价值、债底和风险标签。
- 下修、不强赎、强赎、回售这些公告会影响判断，希望能结构化记录，而不是每天翻公告。
- 单只债需要钻取时，可以直接看隐含波动率、希腊值、价值分解和 σ-S 敏感性热力图。

目前 CBLens 支持：

- Crank-Nicolson PDE 可转债定价
- 强赎、回售、下修博弈、票息、应计利息
- 批量定价和机会分排序
- 巨潮公告事件解析
- Wind / akshare / CSV 数据源
- GUI 桌面 App 和 CLI 命令
- macOS / Windows 桌面包，也可以源码运行

我刻意没有把它做成“买入信号”。里面的机会分、低估率、风险标签都只是帮助缩小复核范围，真正的判断还是要回到条款、公告、流动性和自己的风险承受能力。

### 正文

最近把一个自用项目整理了一版，做成了可以直接跑的桌面工具：CBLens。

项目定位很简单：不是交易软件，不自动下单，也不做投资建议。它更像一个给可转债研究用的工作台，把数据同步、公告事件、准入筛选、理论定价和 GUI 展示放在同一条流程里。

技术上主要是 Python：

- 数值模型：Crank-Nicolson 有限差分 PDE
- GUI：CustomTkinter + matplotlib
- 数据源：Wind、akshare、CSV
- 公告：巨潮 cninfo
- 测试：pytest，目前核心模块有测试覆盖
- 打包：PyInstaller，支持 macOS / Windows 桌面包

功能上主要做这些：

- 单只可转债理论定价、希腊值、债底、转股价值、隐含波动率
- 全市场批量定价，按理论价和市价偏离做排序
- 主池准入筛选：停牌、强赎、ST、临近摘牌、低流动性等
- 公告事件解析：下修、不下修、强赎、不强赎、回售、评级变化
- 敏感性热力图：看正股价格和波动率变化下理论价怎么动

做这个项目的原因也挺朴素：可转债研究里有很多“半结构化”的东西。条款在一个地方，公告在一个地方，行情在一个地方，最后又要回到表格里人工判断。CBLens 想做的是把这些东西先整理成一个可复核的视图。

目前项目已经开源，桌面包也放在 GitHub Releases。感兴趣可以搜：

GitHub: Grefer/CBLens

欢迎体验、提 issue，尤其欢迎反馈：

- 哪些风险标签值得加
- 哪些公告类型解析还不够
- 批量筛选里哪些排序逻辑更符合实际复盘
- Windows / macOS 桌面包有没有兼容问题

最后强调一下：这不是荐股工具，模型价也不是“应该涨到的价格”。它只是把数据和模型偏差更清楚地摆出来，方便自己做下一步判断。

### 正文

大家好，分享一个最近整理出来的开源项目：CBLens。

GitHub:

https://github.com/Grefer/CBLens

它是一个面向 A 股可转债研究和复盘的桌面工作台。项目不做自动交易，也不提供投资建议，主要目标是把条款、公告、行情、模型定价和人工复核流程串起来。

为什么做这个

可转债研究里有很多信息分散在不同地方：

- 转股价、票息、强赎、回售、评级、余额等条款字段
- 下修、不下修、强赎、不强赎、回售等公告事件
- 正股价格、历史波动率、转债市价、股息率、无风险利率
- 批量筛选时还要考虑停牌、ST、临近摘牌、低流动性等状态

我想要的不是一个“买入信号”，而是一个能先把异常标的和模型偏差放大的复核视图，所以就做成了 CBLens。

主要功能

- Crank-Nicolson PDE 可转债定价
- 支持强赎、回售、下修博弈、阶梯票息、应计利息
- 信用利差折现和连续股息率 q
- 单只债希腊值、债底、转股价值、期权溢价、隐含波动率
- 全市场批量定价，输出理论价、市价偏离、低估率、机会分、风险标签
- 主池准入筛选：强赎、停牌、ST、临近摘牌、低成交额、小余额、低评级等
- 巨潮 / Wind 公告事件解析，并可应用回本地条款库
- Wind / akshare / CSV 数据源
- CustomTkinter GUI + CLI
- PyInstaller 桌面包，支持 macOS / Windows

技术栈

- Python 3.10+
- NumPy / SciPy
- CustomTkinter / matplotlib
- akshare
- pdfplumber
- pytest
- PyInstaller

快速开始

```bash
git clone https://github.com/Grefer/CBLens.git
cd CBLens

python -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
pip install -e ".[dev]"

cb-gui
```

没有 Wind 也可以跑离线模型、已有 cb_data、akshare 动态行情；如果要同步完整全市场条款，WindPy 仍然是字段更全的来源。

目前状态

- 核心模块已有 pytest 覆盖
- GUI 能跑完整研究流
- 桌面包可以从 Releases 下载
- 最近补了 CI 和批量缓存并发优化
